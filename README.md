# Nurture

A quick reference for your favourite languages.

Contents

- Small how to compile & run your program
- Hello world example
- Fibonacci example (if present)

## C

```
$ gcc -o {output} {program}.c
$ ./{output}
```

## C++

```
$ g++ -o {output} {program}.cpp
$ ./{output}
```

## Erlang

```
Eshell
1> c({modulename}).
```

## Go

```
$ go run {program}.go
```
Or
```
$ go build {program}.go
$ ./{program}
```

## HTML

```
$ {browser} {page}.html
```

## JavaScript (node)

```
$ node {program}.js
```

## Lisp

```
CL-USER> Invoke krakaboom
```

## Lua

```
$ lua {program}.lua
```

## PHP

```
$ php {program}.php
```

## Python

```
$ python {program}.py
```

## Rust

```
$ rustc {program}.rs
$ ./{program}
```

## Shell

```
$ ./{program}.sh
```
