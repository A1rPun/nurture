## Batch

```
C:\> {program}.bat
```

## C

```
$ gcc -o {output} {program}.c
$ ./{output}
```
Or
```
$ make {program}
$ ./{program}
```

## C++

```
$ g++ -o {output} {program}.cpp
$ ./{output}
```
Or
```
$ make {program}
$ ./{program}
```

## Elixir

```
$ iex {module}.ex

Interactive elixir
iex(1)> {module}.{function}
:ok
```
Or
```
$ elixir -r {module}.ex -e '{module}.{function}'
```

## Erlang

```
$ erl

Eshell
1> c({module}).
{ok,{module}}
2> {module}:{function}().
ok
```

## Go

```
$ go run {program}.go
```
Or
```
$ go build {program}.go
$ ./{program}
```
