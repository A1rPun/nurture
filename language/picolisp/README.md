# PicoLisp

- [Documentation](https://software-lab.de/doc/index.html)
- [Wiki](https://picolisp.com/wiki/?Documentation)

```
$ pil {program}.l
```

### Arguments

```
$ pil {program}.l {argument}
```

### REPL

```
$ pil
: (bye)
$
```

Use `+` at the end of the command to enable debug

## Link a library

```
$ pil lib.l
```
