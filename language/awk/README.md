# AWK

- [Documentation](https://www.gnu.org/software/gawk/manual/gawk.html)
- [Learn AWK in Y minutes](https://learnxinyminutes.com/docs/awk/)
- Indices start at 1

```
$ awk -f {program}.awk
$ awk --bignum -f {program}.awk
```

### One-liner
```
$ awk '{code}'
```

### Shebang
`#!/usr/bin/awk -f`

```
$ chmod +x {program}.awk
$ ./{program}.awk
```
