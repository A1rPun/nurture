# Scala

```
$ scala {program}.scala
```

**One-liner**
```
$ scala -e '{code}'
```



Or create a binary java
```
$ jar cmf META-INF/MANIFEST.MF {program}.jar
$ java -jar {program}.jar
```

Or create a binary
```
$ scalac {program}.scala -d {program}.jar
$ scala {program}.jar
```
