# Python

```
$ python {program}.py
```

### One-liner

```
$ python -c '{code}'
```

### REPL

```
$ python
```

### Shebang

`#!/usr/bin/env python`

```
$ chmod +x {program}.py
$ ./{program}.py
```

### Virtual environment

**create**

```
$ python3 -m venv {directory}
$ source {directory}/bin/activate
```

**delete**

```
$ deactivate
$ rm -r {directory}
```

**install (PyPI)**

```
pip install {package}
```

**list installed**

```
$ pip list
```
